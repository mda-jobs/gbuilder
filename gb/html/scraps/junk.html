The getElementByTagName method returns a NodeList of all elements in the document with the specified element name, in the order in which they would be encountered in a preorder traversal of the Element tree.

For example, this HTML page contains several paragraph elements within the body element. The following code will produce a node list of every p element in the page: 

var nl = document.getElementsByTagName("p");

A nodelist has a length property that can be queried to determine the number of nodes returned by the method as follows: 

  console.log( nl.length);

Each node returned in the previous call can be accessed or visited by traversing the list as follows: 

  for( var i = 0; i < nl.length; i++) {console.log( nl[i]);)

The content (markup and text) of each node can be accessed using the innerHTML property as follows: 

  for( var i = 0, i < nl.length; i++) {console.log( nl[i].innerHTML);)
